# 🐶 Feed The Dog

Oie! Tá aqui o joguinho. Em **Python + Pygame** onde você precisa alimentar os dogs com gostosuras.

## 🎮 Gameplay

Você controla um personagem que joga comida para os cachorros.  
Cuidado! Nem toda comida é boa...
AH, já disse que os cachorros não são muito amigáveis? 👀

## 🕹️ Controles

- ⬅️ ➡️ ⬆️ ⬇️ — Movimentar
- SPACE — Jogar comida
- ENTER — Iniciar o jogo

## 🧠 Regras

- 🍔 Comida boa: **+10 pontos**
- 🥦 Comida ruim: **-10 pontos**
- 🐶 Mordida do cachorro: **-5 pontos**
- ⏱️ Tempo limite para vencer!

## 🎯 Objetivo

Alcance **100 pontos** antes do tempo acabar!

---

## 🚀 Como Jogar (Windows)

1. Acesse a pasta **`dist/`**.
2. Execute o arquivo **`feed_the_dog_v1.0.exe`**.
3. **Aviso:** O Windows pode exibir um alerta de "O Windows protegeu o seu computador". Clique em **"Mais informações"** e depois em **"Executar assim mesmo"**.
